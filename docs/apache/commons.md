# [Apache Commons](https://commons.apache.org)

是一个专注于可重用Java组件的Apache项目。
Apache Commons项目由三部分组成：

- Commons Proper - 一个可重用的Java组件库。
- Commons Sandbox - 一个用于Java组件开发的工作空间。
- Commons Dormant - 一个存储当前处于非活动状态的组件的仓库。

## The Commons Proper（可重用）

Commons Proper致力于一个主要目标：创建和维护可重用的Java组件。Commons Proper是协作和共享的场所，Apache
社区中的开发人员可以在此共同开发项目，并与 Apache 项目和用户分享这些项目。

Commons开发人员将努力确保他们的组件对其他库的依赖最小化，以便这些组件可以轻松部署。此外，Commons组件将尽可能保持其接口稳定，以便
Apache 用户（包括其他 Apache 项目）可以实现这些组件而不必担心未来的更改。

[本文](https://web.archive.org/web/20070105140716/http://www.onjava.com/pub/a/onjava/2003/06/25/commons.html)
概述了可以在这里找到的（一些）组件。

我们欢迎所有感兴趣的人参与，无论技能水平如何。编码、文档和测试都是软件开发过程中至关重要的部分。如果您有兴趣参与其中任何一个方面，请加入我们！

## The Commons Sandbox（沙箱）

Commons项目还包含一个对所有Apache提交者开放的工作区。这是一个尝试新想法并准备将其包含到项目的公共部分或另一个Apache项目中的地方。用户可以自由地尝试在沙箱中开发的组件，但沙箱组件不一定会得到维护，尤其是在其当前状态下。

有关沙箱组件，请参见[此处](https://commons.apache.org/sandbox.html)。

## The Commons Dormant（休眠）

这些Commons组件，它们最近的开发活动很少而被认为是不活跃的。如果您希望使用其中任何一个组件，则必须自己构建它们。最好假设这些组件在不久的将来不会发布。

请参见[此处](https://commons.apache.org/dormant.html)了解休眠组件。

## 我该如何贡献、提供反馈、修复错误等？

Commons项目非常需要并感激任何贡献，包括文档帮助、源代码和反馈。

- 讨论发生在Commons邮件列表上。
- 我们在freenode上有一个IRC频道-加入#apache-commons。
- 可以通过在线和svn客户端访问SVN commons/proper[存储库](https://infra.apache.org/version-control.html)。
- 可以通过在线和svn客户端访问SVN commons/sandbox[存储库](https://infra.apache.org/version-control.html)。
- 可以通过在线和svn客户端访问SVN commons/dormant[存储库](https://infra.apache.org/version-control.html)。
